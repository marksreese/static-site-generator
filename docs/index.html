Cocktail Specs

This website is generated by a personal project to write a static site generator in python that takes local markdown files and converts them to html. The generated html is written to a docs folder that is pushed to my GitHub repo and served by GitHub pages.
If you'd like try to it out, fork the repo, and then write your main page in /content/index.md Additional pages can be generated by other directories in /content/ containing their own index.md Write links to them using markdown link formatting. To view locally, run main.sh in the root directory and open localhost:8888 in a browser. To build for hosting, run build.sh

I've curated a collection of cocktails that I've enjoyed and organized them primarily by base spirit in the following pages. The recipes are first sorted into classics and originals, and then in the order I discovered them, whether online, at a bar, or from a book by a bartender. The name of the associated bar or bartender are next to the cocktail name when possible.

The drinks I make the most often are down below the links.
